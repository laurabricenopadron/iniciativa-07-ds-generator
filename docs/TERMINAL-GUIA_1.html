Guía rápida de Terminal

Referencia de los comandos esenciales para navegar carpetas, usar Git y correr el proyecto.

Abrir la Terminal

La forma más rápida:

# Presionar ⌘ + Espacio → escribir "Terminal" → Enter
Tip: podés fijar Terminal en el Dock para tenerla siempre a mano: click derecho sobre el ícono → Opciones → Mantener en el Dock.

Ir directo al proyecto ⭐

El comando que más vas a usar. Copiá y pegá esto tal cual para entrar a la carpeta del proyecto:

cd "Documents/proyecto 07/iniciativa-07-ds-generator"
Flujo completo desde cero (abrir terminal y llegar al proyecto):

1. ⌘ + Espacio → escribir "Terminal" → Enter
2. Pegar: cd "Documents/proyecto 07/iniciativa-07-ds-generator"
3. Verificar que llegaste bien: pwd (tiene que terminar en iniciativa-07-ds-generator)
Truco anti-errores: si no te acordás la ruta, escribí cd y un espacio, y después arrastrá la carpeta del proyecto desde el Finder hasta la ventana de Terminal. La ruta se completa sola. Enter y listo.
Ojo con los espacios: como la carpeta se llama "proyecto 07" (con espacio), la ruta siempre va entre comillas: cd "Documents/proyecto 07/...". Sin las comillas, la terminal se confunde con el espacio y da error.

Git — flujo diario

Antes de empezar a trabajar

git pull                     # Trae los últimos cambios de GitHub

Después de trabajar

git add .                    # Prepara todos los archivos cambiados
git commit -m "feat: lo que hice"  # Saca la foto con mensaje
git push                    # Sube a GitHub

Ver qué cambié

git status                  # Muestra archivos modificados, nuevos o eliminados

Ver historial

git log --oneline            # Lista de commits resumida
Recordá: en Antigravity también podés hacer todo esto visualmente desde el panel de Source Control, sin usar terminal.

Correr el proyecto

npm install                 # Instala dependencias (primera vez o al agregar nuevas)
npm run dev                 # Arranca el servidor de desarrollo

Después abrir http://localhost:3000 en Chrome.

Para frenar el servidor: Ctrl + C
